USE

Click edit to set the slider. Add, edit, or delete the items you want to display the slider. You can see here the options you have for item:

  • Main Image. Choose the main image.
  • Title. The title of each image.
  • Link. You can add a link in the image.

__________________________________________________________________________________________________________________________________________________________________

You can enter the url of the images manually or press the button "change image". Pressing the button will load the library wordpress images, select the image you want and click "Insert in Post" to insert the image in the slider.

 

After the items, the slider has the configuration options and overall design of the scroll. These are among others: width of the slider, the slider height, colors, ...
